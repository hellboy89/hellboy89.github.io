# 🪙 Cara ou Coroa

> Indeciso? Deixa a sorte decidir.

Aplicação web simples e direta para jogar cara ou coroa. Ideal para resolver dúvidas rápidas, desempatar decisões ou simplesmente tirar na sorte quando você não consegue se decidir.

## Acesse agora

**[hellboy89.github.io](https://hellboy89.github.io/)**

## Preview

![Cara ou Coroa em ação](images/exemplo_uso.png)

## Como usar

1. Abra o link acima no navegador (funciona no celular também)
2. Clique em **Jogar**
3. Aguarde o resultado: **Cara** ou **Coroa**

## Detalhes técnicos

- Resultado gerado com `crypto.getRandomValues()` — aleatoriedade criptográfica, 50/50 garantido
- Sem dependências externas, sem backend, sem instalação
- Funciona em qualquer dispositivo com navegador moderno

## Tecnologias

- HTML5
- CSS3
- JavaScript (Web Crypto API)
